## 3D Printed Parts

All STL files available in the `/STL/` directory on GitHub.

**Filament Required:** ~2.5kg  
**Recommended Material:** PETG or ABS  
**Print Settings:** 0.2mm layer height, 20-30% infill

See the assembly guide for detailed print orientations and support requirements.

- **Local Hardware Stores:** Generic fasteners, wire, zip ties
- **Official Stores:** BIGTREETECH, BIQU direct for electronics
